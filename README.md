

# Conspiser
Данная программа предназначена для поиска пользователей в социальной сети ВКонтакте, по определенным критериям: города, пол и возраст. Пользователи также могут сохранять или блокировать пользователей. 


## Установка и начало работы

### Шаг 1
Для того, чтобы программа работала верно, необходимо загрузить **Python 3.10** и необходимые файлы из *requirements.txt*.

### Шаг 2 
Перед тем как запустить бот, необходимо создать базу данных на своем сервере: 
1. Используйте файл *create_db.py* с вашими данными для создания БД.
2. Открываем *config.py* для создания DSN
3. Используйте *main.py* для сессии
4. Запустите файл *db_models.py*, который автоматически создаст сущностей
